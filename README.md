# tg-math-games

Telegram Mini App — две математические игры на двоих (один экран):

1. **Крестики-нолики 3·3·4** — нестандартное поле, 3 в ряд
2. **Ромашка** — круговой ним, 11/12 фишек, забери последнюю

## Dev

```bash
pnpm install
pnpm dev
```

## Deploy

Push to `main` → Vercel auto-deploys. URL прописывается в @BotFather → `/newapp`.
